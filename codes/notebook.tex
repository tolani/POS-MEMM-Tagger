    \subsection{Feature extraction : to vectorize given
words}\label{feature-extraction-to-vectorize-given-words}

\begin{itemize}
\tightlist
\item
  we are converting each word to a vectorized form which is a binary
  vector ( containing only 0s and 1s )
\item
  Each feature/column is a binary question. e.g. is first letter
  capitalized or not ? ( Yes = 1 / No = 0 )
\item
  We are considering mainly suffix , prefix , prev and next context ,
  some english rules of words as features
\item
  At the end we will store this feature dataframe to a csv file , for
  future use : as this code takes time
\end{itemize}
